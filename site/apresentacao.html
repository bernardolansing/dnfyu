Do Not Forget Your Umbrella

A solução Bluetooth para não esquecer seus pertences

Apresentado por:

Augusto Kessler Pires

Bernardo Lansing

Sofia Popsin Gomes

Thiago Trautwein Santos

O Problema

  • - Esquecer objetos pessoais é algo comum, principalmente em rotinas corridas.
  • - Guarda-chuvas, mochilas e bolsas são frequentemente deixados para trás.
  • - Faltava uma solução simples e automática para esse tipo de esquecimento.

A Solução

  • - Dispositivo Bluetooth de baixa energia (BLE) instalado no objeto pessoal.
  • - Aplicativo móvel que monitora a proximidade via sinal BLE.
  • - Notificações são enviadas ao usuário ao se afastar do objeto.

Tecnologias Utilizadas

  • - ESP32 com ESP-IDF: responsável por emitir sinais BLE.
  • - Rust (esp32-nimble): usado para configurar e controlar os anúncios Bluetooth.
  • - Android com Jetpack Compose: app que detecta e interpreta o sinal.
  • - Kotlin + Android SDK: para interface e notificações.

Funcionamento (ESP32)

  • - Define UUID exclusivo do serviço Bluetooth.
  • - Configura os dados de anúncio (appearance code = "tag").
  • - Define intervalo de broadcast entre 20ms e 40ms.
  • - Mantém emissão contínua do sinal BLE.

Funcionamento (App Android)

  • - Detecta sinais BLE com UUID esperado.
  • - Mede intensidade do sinal (RSSI).
  • - Se o sinal enfraquecer abaixo de um limiar → dispara alerta.
  • - Interface simples com foco em acessibilidade e rapidez.

Resultados Esperados

  • - Redução de esquecimentos em ambientes públicos.
  • - Solução reutilizável com diferentes tipos de objeto.
  • - Uso eficiente de bateria com BLE.

Futuro e Melhorias

  • - Adicionar GPS como fallback em locais sem sinal BLE.
  • - Suporte a múltiplos dispositivos por app.
  • - Registro de histórico de alertas.

Conclusão

  • - Usa tecnologias acessíveis e de baixo consumo.
  • - Pode ser expandido para diversos casos de uso além de guarda-chuva.
